# Customer Churn Prediction (MLDP Project)

This project applies supervised machine learning techniques to predict customer churn
in a telecommunications company.

## Problem Statement
Customer churn negatively impacts business revenue. This project aims to predict
whether a customer is likely to churn based on their service usage and account information.

## Technologies Used
- Python 3
- scikit-learn
- pandas, numpy
- Streamlit
- GitHub (Version Control)

## Dataset
Telco Customer Churn Dataset (Kaggle)
